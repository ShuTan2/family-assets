# 家庭资产记账APP - 产品需求文档

## 1. 产品概述

家庭资产记账APP是一款移动端应用，帮助用户管理家庭银行存款和定期存款，提供直观的资产总览和到期提醒功能。

### 主要目标
- 记录每一笔存款的详细信息
- 按到期日排序展示，临近到期高亮提醒
- 提供资产分布图表，总览家庭财务状况

### 目标用户
- 有家庭理财需求的用户
- 希望追踪定期存款到期日的用户

---

## 2. 核心功能

### 2.1 存款记录管理

**支持的存款类型：**
- 银行名称
- 存款类型（定期/活期）
- 年化利率
- 存期
- 预计到期收益
- 起存日期
- 到期日期（自动计算）

**功能要求：**
- 添加存款记录
- 编辑存款记录
- 删除存款记录
- 按到期日由近到远自动排序

### 2.2 到期提醒高亮显示

| 条件 | 高亮颜色 | 说明 |
|------|----------|------|
| 30天内到期 | 红色 | 紧急提醒 |
| 30-90天内到期 | 橙色 | 提前关注 |

### 2.3 资产总览

**总资产统计：**
- 总资产金额
- 活期存款总额
- 定期存款总额
- 总预期收益

**图表展示：**
- 总资产概览卡片
- 银行分布饼图
- 存款类型分布饼图（定期/活期）
- 存期分布柱状图

---

## 3. 页面设计

### 3.1 页面结构

| 页面名称 | 功能描述 |
|----------|----------|
| 首页/资产总览 | 显示总资产、各类统计卡片、分布图表 |
| 存款列表 | 展示所有存款记录，按到期日排序，带高亮 |
| 添加/编辑存款 | 存款信息表单 |

### 3.2 设计风格

**视觉方向：**
- 简约现代的金融风格
- 数据可视化清晰易读
- 移动端优先，触摸友好

**色彩方案：**
- 主色：深蓝色 #1E3A5F
- 辅助色：浅蓝 #4A90D9
- 强调色：金色 #D4AF37
- 警示红：#E74C3C
- 警示橙：#F39C12
- 背景色：#F8F9FA
- 文字色：#2C3E50

**字体：**
- 标题：思源黑体 / Noto Sans SC Bold
- 正文：思源黑体 / Noto Sans SC Regular
- 数字：DIN Alternate / Roboto Mono

**布局：**
- 底部导航栏（首页、存款、添加）
- 卡片式信息展示
- 图表可视化

---

## 4. 交互设计

### 4.1 首页/资产总览
- 顶部显示总资产金额
- 中部四个统计卡片（活期总额、定期总额、笔数、即将到期）
- 底部三个图表（银行分布、类型分布、存期分布）

### 4.2 存款列表
- 列表项显示：银行、类型、金额、到期日、状态
- 红色/橙色高亮背景表示即将到期
- 点击进入详情/编辑
- 右滑删除

### 4.3 添加/编辑存款
- 表单输入：银行、类型、金额、利率、存期、起存日期
- 自动计算到期日期和预计收益
- 表单验证

---

## 5. 技术说明

**前端技术：**
- React + TypeScript
- TailwindCSS
- Recharts（图表库）
- Vite（构建工具）
- LocalStorage（数据持久化）

**数据存储：**
- 使用浏览器LocalStorage存储存款数据
- 支持数据导入导出（JSON格式）
